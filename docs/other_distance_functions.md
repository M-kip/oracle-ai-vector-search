# Other Distance Functions

- ```L1_DISTANCE(v1, v2)```
    - i.e. Manhattan Distance
- ```L2_DISTANCE(v1, v2)```
    - i.e. Euclidean Distance
    - shorthand: ```v1 <-> v2```
- ```COSINE_DISTANCE(v1, v2)```
    - i.e. Cosine Distance
    - shorthand: ```v1 <=> v2```
- ```INNER_PRODUCT(v1, v2)```
    - i.e. Dot Product Distance
    - shorthand (negative): ```v1 <#> v2```
